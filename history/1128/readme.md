깃으로 무엇을 할 수 있을까?
1️⃣ 버전 관리

문서를 수정할 때마다 언제, 어떤 내용이 변경되었는지 구체적으로 기록할 수 있음

이러한 기능을 제공하는 버전 관리 시스템이 바로 Git

2️⃣ 백업

현재 컴퓨터의 자료를 다른 컴퓨터에 복제하는 것

깃 파일을 위한 백업 공간을 제공하는 인터넷 서비스를 원격 저장소 또는 온라인 저장소라고 함

가장 많이 사용되는 서비스는 깃허브(GitHub)

3️⃣ 협업

깃허브 같은 온라인 서비스를 통해 여러 사람이 파일을 함께 수정하고 공유 가능

누가, 어떤 부분을 수정했는지 기록으로 남기므로 오류 발생 시 원인 파악이 쉬움

협업 중 발생하는 문제를 정리하고 관리하는 기능도 포함됨

💡 3단계 학습 순서

버전 관리 → 백업 → 협업
버전 관리를 이해하지 못하면 백업을 이해할 수 없고, 백업을 모르면 협업도 불가능함.

깃 프로그램의 종류
🖥️ 깃허브 데스크톱 (GitHub Desktop)

깃허브에서 제공하는 공식 프로그램 → https://desktop.github.com

GUI(그래픽 인터페이스) 기반이라 사용이 쉽고 입문자에게 적합

기본 기능 중심이라 고급 사용자는 다소 부족하게 느낄 수 있음

🐢 토터스깃 (TortoiseGit)

윈도우 탐색기 메뉴에 통합되는 윈도우 전용 Git 프로그램

다운로드: https://tortoisegit.org/download/

파일 탐색기에서 바로 Git 명령을 실행할 수 있어 편리함

🌳 소스트리 (SourceTree)

Git의 기본부터 고급 기능까지 모두 사용 가능한 프로그램

다운로드: https://www.sourcetreeapp.com/

기능이 많아 복잡하지만, 익숙해지면 깃을 가장 자유롭게 활용할 수 있음
