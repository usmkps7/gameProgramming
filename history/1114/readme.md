## Unity Commands Used

### Project Creation
- New project
- 3D Core Template
- Create project

### Hierarchy Operations
- Create → 3D Object → Cube / Sphere
- Duplicate (Ctrl + D)
- Rename
- Drag & Drop Materials / Physic Materials

### Scene Controls
- Alt + Drag (Orbit view)
- Ctrl + Alt + Drag (Pan)
- Mouse Wheel (Zoom)
- F key (Frame selected)
- Perspective ↔ Isometric toggle

### Inspector → Transform
- Position (X,Y,Z)
- Rotation (X,Y,Z)
- Scale (X,Y,Z)

### Component Menu
- Component → Physics → Rigidbody
- Rigidbody options: Use Gravity, Mass, Drag, Angular Drag

### Materials
- Create → Material
- Change Albedo color
- Apply Material by drag & drop

### Physic Materials
- Create → Physic Material
- Set Bounciness
- Apply via Collider → Material
